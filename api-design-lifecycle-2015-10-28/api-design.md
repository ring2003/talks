## Nejčastější chyby

- Nekonzistentnost
- Nedodržování konvencí (např. HTTP kódy)
- Nesrozumitelné chybové hlášky
- Mizerná pokud nějaká dokumentace

- Zájmy server vývojáře se významně liší od klient vývojáře
